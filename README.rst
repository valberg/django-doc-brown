Supported versions
------------------

- Python 3.6, 3.7, 3.8
- Django 2.2.x, 3.0.x

Current state
-------------

django-model-revisioning is currently not production ready - we need more tests for that!

You can install the pre-release from PyPI though using::

    $ pip install django-model-revisioning
